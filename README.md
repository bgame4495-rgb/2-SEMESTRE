# 2-SEMESTRE

## comandos basicos - GIT

1. Trazendo possiveis mudanças do GitHub:
git pull

2. adicionar arquivos para commit:
git add .

3. criar versionamento (commit):
git commit ~m "comandos basicos do git."

4. Enviar alterações para GitHub:
git push


## Branch

Mostrar Branche no computador:
git branch a
